# Expense Tracker
**Автор:** Ярин Егор

## Описание
Простая программа для контроля личных финансов. Позволяет записывать траты, классифицировать их и фильтровать историю.

## Как использовать
1. Запустите `python main.py`.
2. Введите сумму расхода (число).
3. Выберите или введите категорию.
4. Укажите дату (по умолчанию стоит сегодняшняя).
5. Нажмите «Добавить расход».

## Особенности
- Сохранение данных в JSON.
- Валидация ввода.
- Фильтрация по категориям и автоматический подсчет суммы.
